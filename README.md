### Mini-Game Spotlight: Decathlon Challenge (Rhythm & Animation)

#### Concept
This mini-game was created as part of the **Decathlon Challenge (Level 3)**, requiring an animation-based experience.
The game begins with a questionnaire that determines the player‚Äôs profile. Based on the player‚Äôs answers, the mascot bee performs different workout animations (push-ups, lunges, strength training, etc.).

#### Gameplay
After the questionnaire, the game transforms into a short FnF-style rhythm challenge:
* Press the right keys at the right moment
* Follow the beat of a CC0-licensed song
* Enjoy custom animations and visuals tied to your choices

#### Notable Features
* Fully custom UI and character animation using CSS & JS
* AI-generated background, but all other designs drawn manually
* Dynamic behavior depending on player choices
* A lightweight rhythm-game engine coded from scratch

### Mini-Game Spotlight: CVE Fishing (Security Meets Timing)

#### Concept
This mini-game introduces cybersecurity concepts through a playful fishing mechanic.
Each ‚Äúfish‚Äù is actually a **CVE (Common Vulnerabilities and Exposures)**, each with its own rarity tier and description.

#### Gameplay
Inspired by Undertale‚Äôs attack bar system, the player must:

#### Objective & Message
By comparing the volume and danger level of CVEs, the game subtly highlights the contrast between Windows and Linux in terms of vulnerability management.

#### Notable Features
* 100% JavaScript-driven game logic
* Simple but effective timing mechanics inspired by classic RPGs
* Informative CVE cards that appear after each catch
* Rarity system to keep gameplay varied and engaging
