# NIRD – base web Nuit de l'Info

Projet statique en HTML/CSS/JS pour présenter les défis Nuit de l'Info autour du NIRD (défense des droits numériques, promotion de Linux et du libre).

## Pages
- `index.html` : page d'accueil NIRD, navigation vers les défis.
- `femmes-info.html` : espace pour vidéo/image/texte du défi Femmes & info.
- `decathlon.html` : aperçu des épreuves du défi Décathlon.
- `cve-explorer.html` : formulaire pour interroger une CVE via l'API publique `cve.circl.lu`.
- `ergonomie.html` : démonstrateur de dark patterns autour d'un faux formulaire.

## Utilisation
Ouvrez simplement `index.html` dans un navigateur. Les autres pages sont accessibles via le menu.
